Question-1: What are the different ways to select an element in the DOM?

To select an element, it depends mostly on the user's need. Like, let's say the user wants to specifically select ‘an’ element here, he can use a single element selector such as getElementById. If he wants to select multiple elements, then there are a lot more approaches: let’s say he wants to select an element that shares a common class, here getElementsByClassName can be used. Or getElementsByTagName can be used to select elements with the same tag name.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML is used to change the whole structure of a selected element. innerText is used to change the text inside of a selected element. textContent works similarly to innerText, but it can also change hidden elements.

Question-3: What is event delegation in the DOM?

It is a bubbling technique where a developer can add a single event listener to a parent element rather than adding multiple event listeners to the children.

Question-4: What is event bubbling in the DOM?

Event bubbling refers to the HTML structure being an overly complicated network of parent and children elements, where a parent element (div, main, section, ul) can have multiple children (sub div, li, etc.).

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, we use document.createElement, which instructs the HTML document to create a new element. Add function is used to add a child to a selected parent. Remove is used to remove an element from a selected parent.
